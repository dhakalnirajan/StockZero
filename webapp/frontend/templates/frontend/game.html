StockZero Chess AI

About StockZero

StockZero is a Reinforcement Learning based chess engine, combining strategic insights with tactical precision. Play against StockZero and experience a challenging chess game powered by AI.

This web application is built using Django, TensorFlow, and Chessboard.js. The AI engine is trained using self-play and Monte Carlo Tree Search, inspired by AlphaZero and Stockfish methodologies.

Enjoy your game!
